Laravel Routing

Routing is one of the fundamental concepts in Laravel. Routes are defined in routes/web.php for web routes and routes/api.php for API routes.

Basic Route Definition

Route::get('/welcome', function () {
    return 'Hello, World!';
});

This creates a route that responds to GET requests at /welcome URL.

Common Route Methods

  • Route::get() - GET requests
  • Route::post() - POST requests
  • Route::put() - PUT requests
  • Route::delete() - DELETE requests
  • Route::any() - Any HTTP method

Route Parameters

Route::get('/user/{id}', function ($id) {
    return 'User '.$id;
});

Route::get('/post/{post}/comment/{comment}', function ($postId, $commentId) {
    // Multiple parameters
});
