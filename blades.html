Laravel Blade Templates

Blade is Laravel's simple yet powerful templating engine. Unlike other PHP templating engines, Blade does not restrict you from using plain PHP code in your views.

Basic Blade Syntax

Blade files use the .blade.php file extension and are typically stored in the resources/views directory.

<!-- Displaying data -->
<h1>Hello, {{ $name }}!</h1>

<!-- Conditional statements -->
@if (count($records) === 1)
    I have one record!
@elseif (count($records) > 1)
    I have multiple records!
@else
    I don't have any records!
@endif

<!-- Loops -->
@foreach ($users as $user)
    <p>This is user {{ $user->id }}</p>
@endforeach

Template Inheritance

Blade's template inheritance allows you to build a master template that can be extended by child views.

Master Layout (layout.blade.php)

<html>
    <head>
        <title>App Name - @yield('title')</title>
    </head>
    <body>
        @section('sidebar')
            This is the master sidebar.
        @show

        <div class="container">
            @yield('content')
        </div>
    </body>
</html>

Child View (home.blade.php)

@extends('layout')

@section('title', 'Page Title')

@section('sidebar')
    @@parent
    <p>This is appended to the master sidebar.</p>
@endsection

@section('content')
    <p>This is my body content.</p>
@endsection

Common Blade Directives

  • @if / @else / @endif - Conditional statements
  • @forelse / @empty / @endforelse - Looping through arrays with an else clause
  • @foreach / @endforeach - Looping through arrays
  • @for / @endfor - For loops
  • @include - Including sub-views
  • @extends - Extending a layout
  • @section / @yield - Defining and yielding sections
