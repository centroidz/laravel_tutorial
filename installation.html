Laravel Installation Guide

Follow these steps to get Laravel installed on your system and create your first project.

Prerequisites

  • PHP 8.1 or higher
  • Composer (PHP dependency manager)
  • Database (MySQL, PostgreSQL, SQLite)
  • Web server (Apache/Nginx) or PHP built-in server

Step 1: Install Composer

If you don't have Composer installed, download it from getcomposer.org

Step 2: Create New Laravel Project

composer create-project laravel/laravel my-project

This command will create a new Laravel project in a "my-project" directory.

Step 3: Serve Your Application

cd my-project
php artisan serve

Your application will be available at http://localhost:8000

Once installed, you should see the default Laravel welcome page. Ready to learn about routing?

Having trouble? Check the resources page for help.
